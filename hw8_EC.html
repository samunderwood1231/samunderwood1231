Explanation of Script

The demo above makes an API call to the FreeCocktailDB. I referenced code from Public APIs: Using TheCocktailDB API with JavaScript as well as from Using a Fetch APIâ€¦To Make Cocktails (Medium article) to assebmle the demo on this page. The user inputs an ingredient into the text input field. The submit button uses onclick() to call the function ingredientLookup(), which makes a fetch call to the FreeCocktailDB API. The result is returned from the API as an object. The getIngredientInfo() function returns the .strDescription property of the object. Finally, the string is printed in a div element. Some observations: if a user passes the empty string, the API returns the results for "gin". I was able to get around this by adding data validation before sending the API call. One thing I could not figure out how to do was validate the data that was returned from the server, for example, if the return value === null, print "The ingredient was not found". That would be one improvement to make to this page.
